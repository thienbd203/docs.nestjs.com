- Thư mục này chứa các bản dịch tiếng Việt song song với bản tiếng Anh
- Mỗi file trong thư mục cha (`../`) có bản dịch tương ứng trong thư mục này
- Cấu trúc thư mục được giữ nguyên để dễ dàng đối chiếu

## Đóng góp

Nếu bạn muốn đóng góp vào bản dịch, vui lòng:
1. Fork project này
2. Tạo branch mới cho bản dịch
3. Thêm hoặc cập nhật file dịch
4. Submit pull request

## Lưu ý

- Giữ nguyên format markdown và cấu trúc file
- Code examples nên giữ nguyên tiếng Anh
- Thuật ngữ kỹ thuật quan trọng nên giữ nguyên hoặc có note giải thích
